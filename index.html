Paso 1

Importación y Limpieza

El primer paso es crucial. Un texto limpio es la base de un EPUB profesional. Esto implica eliminar todos los formatos innecesarios y estandarizar la estructura del documento.

  • Eliminar saltos de línea y espacios repetidos.
  • Asegurar un flujo de texto continuo y sin páginas en blanco.
  • Evitar cualquier tipo de formato manual que pueda interferir con los estilos.
Paso 2

Definición de Estilos

La creación de estilos de párrafo es fundamental. Define estilos para cada elemento (títulos, cuerpo, etc.) para mantener la consistencia y facilitar la exportación.

  • Estilo Título: Configura fuente, espaciado y alineación.
  • Estilo Cuerpo: Define la tipografía principal, justificación e idioma.
  • Opción Clave: Marca "Conservar líneas juntas" para evitar viudas y huérfanas.
  • Separación de Sílabas: Actívala para un flujo de texto más natural.
Paso 3

Aplicación Masiva

Aplica los estilos creados de forma eficiente. Utiliza herramientas como "Buscar/Cambiar" con expresiones GREP para automatizar la asignación de estilos a lo largo de todo el documento.

  • Usa Scripts para tareas repetitivas.
  • Con GREP, puedes buscar patrones (ej: todo texto en mayúsculas) y aplicarles el estilo "Título" automáticamente.
  • Esta técnica ahorra horas de trabajo manual y garantiza cero errores.
Paso 4

Tabla de Contenido

La tabla de contenido (TOC) es obligatoria y esencial para la navegación del EPUB. Se genera automáticamente a partir de los estilos de párrafo que has definido.

  • Crea un nuevo estilo de TOC desde el menú "Maquetación".
  • Selecciona el estilo de párrafo que conformará la TOC (ej: "Título").
  • ¡Crítico!: Marca siempre la opción "Crear anclaje de texto en párrafo de origen" para que los links funcionen.
Paso 5

Etiquetas de Exportación

Antes de exportar, debes indicarle al programa cómo convertir tus estilos en código HTML. Esto se hace a través de las "Etiquetas de Exportación" en las opciones de cada estilo.

  • Edita el estilo "Título" y en "Etiquetas de exportación", asigna la etiqueta HTML `h2`.
  • Activa la opción "Dividir documento (solo EPUB)" para que cada capítulo empiece en una nueva página.
  • Asigna etiquetas semánticas a otros estilos (ej: `blockquote` para entradas).
Paso 6

La Exportación Final

Es el momento de generar el archivo EPUB. Configura correctamente las opciones de exportación para asegurar un resultado óptimo y compatible.

  • Formato: Elige EPUB (ajustable), versión 3.0.
  • Portada: Selecciona la imagen de portada.
  • Navegación: Usa el estilo de TOC que creaste.
  • División: Basa la división del documento en las "etiquetas de exportación".
  • Metadatos: Rellena toda la información del libro (título, autor, etc.).
Paso 7

Ajustes en Código

Un EPUB profesional casi siempre requiere ajustes finos en su código. Usando un editor como Calibre (gratuito), puedes modificar directamente el HTML y CSS para un control total.

  • Abre el EPUB en un editor de código como Calibre.
  • Navega a la hoja de estilos CSS.
  • Modifica las propiedades de las clases generadas (ej: cambia el color de todos los `h2`).
  • La edición directa del CSS te permite lograr efectos visuales imposibles desde el software de maquetación.
